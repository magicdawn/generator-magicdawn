# generator-magicdawn

[![Greenkeeper badge](https://badges.greenkeeper.io/magicdawn/generator-magicdawn.svg)](https://greenkeeper.io/)
Yeoman generator for self use

## Install

```
npm i generator-magicdawn -g
```

## Run

- new repo on GitHub, no `.gitignore`, no `LICENSE`
- `npm init` fill `name` `version` `description`
- `yo magicdawn`
